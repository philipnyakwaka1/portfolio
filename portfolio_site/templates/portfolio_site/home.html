Portfolio Application

Django RESTful Portfolio Management System

Features

  • RESTful API endpoints for portfolio data
  • Modular Django app architecture
  • Projects, Skills, Experience, and Contact management
  • Django Admin interface for content management
  • Static files and media serving support

Available API Endpoints

GET /api/projects/

List all portfolio projects. Supports ?featured=true filter.

GET /api/skills/

List all skills. Supports ?category=Backend filter.

GET /api/experience/

List all work experience entries.

GET /api/contact/

List contact submissions (admin only).

POST /api/contact/

Submit a new contact form message.
